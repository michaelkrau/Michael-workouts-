# הנחיות בטיחות
# Safety Guidelines

## עקרונות בטיחות כלליים / General Safety Principles

### לפני כל אימון / Before Every Workout

✅ **בדיקת בריאות**
- שאלי על בעיות בריאותיות
- וודאי שאין הריון (ללא אישור רפואי)
- בדקי פציעות קיימות
- וודאי שאין כאבים חריגים

✅ **הכנת מרחב**
- שטח נקי ופנוי
- תאורה מספקת
- רצפה לא חלקה
- מרחק בטיחות בין משתתפים

✅ **בדיקת ציוד**
- ציוד תקין וללא פגמים
- גומיות ללא סדקים
- כדורים מנופחים כראוי
- משקולות יציבות

✅ **לבוש מתאים**
- בגדי ספורט נוחים
- נעלי ספורט עם אחיזה
- שיער קשור
- תכשיטים הוסרו

---

## חימום והתקררות / Warm-up & Cool-down

### חימום (10-15 דקות) - חובה!

**מדוע חשוב:**
✓ מעלה טמפרטורת שרירים
✓ מגביר זרימת דם
✓ מכין את המפרקים
✓ מונע פציעות
✓ משפר ביצועים

**מבנה חימום:**
1. **אירובי קל** (5 דקות) - מעלה דופק
2. **מוביליות מפרקים** (3 דקות) - תנועות עגולות
3. **מתיחות דינמיות** (2 דקות) - תנועות פעילות
4. **אקטיבציה ספציפית** (3 דקות) - תרגילי הכנה

### התקררות (5-10 דקות) - חובה!

**מדוע חשוב:**
✓ מורידה דופק בהדרגה
✓ מפחיתה כאבי שרירים
✓ מסייעת להתאוששות
✓ מונעת סחרחורת

**מבנה התקררות:**
1. **אירובי קל** (2 דקות) - הליכה איטית
2. **מתיחות סטטיות** (5 דקות) - 30 שניות כל שריר
3. **נשימות עמוקות** (2 דקות) - הרפיה

---

## סימני אזהרה - עצרי מיד! / Warning Signs - Stop Immediately!

### 🚨 הפסקי את האימון אם:

**כאב:**
- כאב חד או עוקץ
- כאב בחזה
- כאב במפרקים
- כאב בגב תחתון

**סחרחורת:**
- סחרחורת חזקה
- בחילה
- הרגשה של עילפון
- אובדן איזון

**נשימה:**
- קוצר נשימה חריג
- קושי לנשום
- אי נוחות בנשימה

**אחר:**
- דופק לא סדיר
- חולשה פתאומית
- עייפות קיצונית
- הרגשה "משהו לא בסדר"

**במקרה כזה:**
1. עצרי את האימון מיד
2. שבי ונוחי
3. שתי מים
4. אם לא עובר - פני לעזרה רפואית

## טכניקה נכונה / Proper Technique

### עקרונות יסוד / Basic Principles

**יציבה:**
✓ עמוד שדרה ישר
✓ כתפיים למטה ואחורה
✓ בטן מופעלת
✓ ברכיים רכות (לא נעולות)

**נשימה:**
✓ אל תעצרי נשימה!
✓ נשמי החוצה במאמץ
✓ נשמי פנימה בחזרה
✓ נשימה עמוקה ושלוטה

**שליטה:**
✓ תנועות מבוקרות
✓ אין תנופה
✓ טווח תנועה מלא
✓ Tempo קבוע

**התקדמות:**
✓ התחילי קל
✓ עלי בהדרגה
✓ איכות לפני כמות
✓ הקשיבי לגוף

## בטיחות לפי ציוד / Equipment-Specific Safety

### משקולות / Dumbbells
- התחילי עם משקל קל
- בדקי אחיזה טובה
- הורדה מבוקרת
- אל תשליכי משקולות

### קטלבל / Kettlebell
- למדי טכניקה נכונה תחילה!
- שמרי מרחק מאחרים
- אחיזה חזקה
- הכוח מהירכיים, לא הגב

### גומיות / Resistance Bands
- בדקי סדקים לפני שימוש
- אל תמתחי יותר מדי
- שמרי מרחק בטיחות
- השתמשי באחיזה נכונה

### כדור פיזיו / Stability Ball
- בדקי ניפוח נכון
- שטח לא חלק
- התחילי ליד קיר
- תמיד יד על הכדור

---

## בטיחות למצבים מיוחדים / Safety for Special Conditions

### הריון / Pregnancy
⚠️ דורש אישור רפואי!
- אימונים מותאמים בלבד
- אין שכיבה על גב (טרימסטר 2+)
- אין קפיצות/זעזועים
- הקשיבי לגוף

### פציעות קיימות / Existing Injuries
- התייעצי עם רופא/פיזיותרפיסט
- התאימי תרגילים
- הימנעי מתנועות כואבות
- התקדמי לאט

### מבוגרים / Seniors
- קצב איטי יותר
- התמקדי באיזון
- הרבה מנוחה
- תרגילי כוח עדינים

### בעיות לב / Heart Conditions
- אישור רפואי חובה!
- עקבי אחרי דופק
- הימנעי מעצימות גבוהה
- עצרי אם צריך

---

## תגובה לחירום / Emergency Response

### מה לעשות בחירום / What to Do in Emergency

**1. עצרי את האימון מיד**

**2. הערכי את המצב:**
- האדם בהכרה?
- נושם?
- דופק?

**3. פעלי לפי הצורך:**
- התקשרי ל-101 אם דרוש
- בצעי עזרה ראשונה אם יודעת
- הישארי עם האדם
- הרגיעי אותו

**4. תעדי:**
- מה קרה
- מתי
- מה עשית
- איך הסתיים

### ערכת עזרה ראשונה / First Aid Kit

**חובה בכל מקום אימון:**
- תחבושות
- גזה סטרילית
- פלסטרים
- אלכוהול לחיטוי
- כפפות חד-פעמיות
- מספריים
- מספרי חירום (101, הצלה)

## אחריות מקצועית / Professional Responsibility

### כמדריכה, את אחראית על:

✓ **הכשרה מתאימה** - הסמכה מקצועית
✓ **ביטוח** - ביטוח אחריות מקצועית
✓ **עזרה ראשונה** - קורס תקף
✓ **עדכון ידע** - הדרכות מתמשכות
✓ **תיעוד** - רישום משתתפים ובעיות בריאות
✓ **תקשורת** - הסברים ברורים
✓ **השגחה** - עקיבה אחרי כולם

### מסמכים נדרשים:

- [ ] הסמכת מדריכת כושר
- [ ] ביטוח אחריות מקצועית
- [ ] אישור עזרה ראשונה
- [ ] הצהרות בריאות משתתפים
- [ ] כתב ויתור (אם נדרש)

## צ'קליסט בטיחות / Safety Checklist

### לפני כל אימון / Before Every Workout:
- [ ] בדקתי שטח האימון
- [ ] בדקתי את הציוד
- [ ] אספתי מידע בריאותי
- [ ] הכנתי ערכת עזרה ראשונה
- [ ] תכננתי חימום והתקררות

### במהלך האימון / During Workout:
- [ ] עוקבת אחרי טכניקה
- [ ] שמה לב לסימני עייפות
- [ ] מתאימה לרמות שונות
- [ ] מעודדת הפסקות מים
- [ ] זמינה לשאלות

### אחרי האימון / After Workout:
- [ ] וידאתי שכולם בסדר
- [ ] סידרתי את הציוד
- [ ] תיעדתי אירועים חריגים
- [ ] תכננתי שיפורים
